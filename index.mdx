## 📘 Documentação oficial da API Mentor IA

Bem-vindo à **documentação oficial da API do Mentor IA**, a plataforma inteligente da RD Station que atua como seu “segundo cérebro”: gera conteúdo, qualifica leads e automatiza atendimento com uma voz autêntica e humanizada.

---

## Módulos disponíveis na API

A API do Mentor IA permite acesso aos seguintes módulos:

- **Meu Assistente** 🧠  
  Widget virtual para interação com visitantes do site, qualificação de leads e atendimento automatizado com base em conhecimento estruturado.

- **Contas** 🧾  
  Gerencie e consulte as contas conectadas ao Mentor IA, incluindo autenticações e configurações de integração.

- **Conteúdo** 📚  
  Obtenha e organize conteúdos gerados para blogs, redes sociais, e-mails, anúncios e outros.

- **Base de Conhecimento** 🔍  
  Acesse, atualize e expanda os artigos que alimentam as respostas do assistente, garantindo informação precisa e relevante.

---

## O que é o Mentor IA?

O Mentor IA é uma solução da RD Station que combina geração de conteúdo e assistentes virtuais treinados com dados do seu negócio, permitindo:

- Criação acelerada de conteúdo para blogs, redes sociais, anúncios, e-mails, etc.
- Assistência virtual humanizada que aprende com seus dados e conversa como você

---

## Autenticação

Todas as requisições devem incluir um **Bearer Token** no cabeçalho HTTP:

```http
Authorization: Bearer SEU_TOKEN_AQUI
```
