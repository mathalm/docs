## 📘 Documentação Oficial da API do Mentor IA

Bem-vindo à **documentação oficial da API do Mentor IA**, a plataforma inteligente da RD Station que atua como seu “segundo cérebro”: gerando conteúdo, qualificando leads e automatizando atendimentos com uma voz autêntica e humana.

---

## Módulos Disponíveis na API

A API do Mentor IA oferece acesso aos seguintes módulos:

- **Meu Assistente** 🧠  
  Um widget virtual para interagir com visitantes do site, qualificar leads e realizar atendimentos automáticos com base em conhecimento estruturado.

- **Contas** 🧾  
  Gerencie e consulte as contas conectadas ao Mentor IA, incluindo configurações de autenticação e integração.

- **Conteúdo** 📚  
  Recupere e organize conteúdos gerados para blogs, redes sociais, e-mails, anúncios e muito mais.

- **Base de Conhecimento** 🔍  
  Acesse, atualize e expanda os artigos que alimentam as respostas do assistente, garantindo informações precisas e relevantes.

---

## O que é o Mentor IA?

O Mentor IA é uma solução da RD Station que combina geração de conteúdo e assistentes virtuais treinados com os dados do seu negócio, possibilitando:

- Criação acelerada de conteúdo para blogs, redes sociais, anúncios, e-mails etc.
- Assistência virtual com linguagem humana que aprende com seus dados e fala como você

## Autenticação

<AccordionGroup>
  <Accordion icon="key" title="Obtenha sua chave de API">
    Pegue sua chave de API em **Contas -> Configurações -> Chaves de API** – ela
    é necessária para autenticação.
  </Accordion>
</AccordionGroup>

Todas as requisições devem incluir um **Bearer Token** no cabeçalho HTTP:

```http
Authorization: Bearer SEU_TOKEN_AQUI
```
